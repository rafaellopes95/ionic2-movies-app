# Ionic 2 - Movies App

This is a simple app developed with Ionic 2.

It uses an API to get movies information.

# Dependencies

First make sure you have installed NodeJS 6.7 or higher.

Then you can install Ionic 2 by executing the below in terminal:
   
```
npm install -g ionic
```

Finally, to download the app dependencies (modules), open a terminal window inside app project folder and execute the below:

```
npm install
```
